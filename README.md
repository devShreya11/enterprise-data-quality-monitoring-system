# Enterprise Data Quality Monitoring System

## Project Overview

This project simulates a real-world enterprise data quality monitoring solution for Material Master data.

The system automatically identifies common data quality issues such as duplicate records, missing values, invalid units, incorrect categories, and pricing anomalies using SQL, Python, and Power BI.

## Business Problem

Poor quality master data leads to:
- Duplicate inventory
- Procurement errors
- Incorrect reporting
- Increased operational costs

This project demonstrates how automated data validation improves enterprise data quality.
